# dashboard-api
Express JS

Demo del proyecto [https://jjhoncv.github.io/dashboard](https://jjhoncv.github.io/dashboard/) 

## Features

- Version de express 4
- Version de Typescript 4

### step 1
dependencies
```sh
make install
```

### step 2
up mysql
```sh
make db
```

### step 3
up express api
```sh
make start
```

### extras
down express api
```sh
make stop
```
