# dashboard-api
Express JS

Demo del proyecto [http://wdashboard.tk](http://wdashboard.tk) 

## Features

- EC2 server
- RDS MySQL
- Version de Express JS 4
- Version de Typescript 4
- Version de MySQL 5.6

### Step 1
Up MySQL
```sh
make db
```

### Step 2
Up Express JS - API
```sh
make start
```

### Extras
Down Express JS - API
```sh
make stop
```

Dependencies NPM
```sh
make install
```
