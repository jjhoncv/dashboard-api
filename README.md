# dashboard-api
Express JS

Demo del proyecto [https://jjhoncv.github.io/dashboard](https://jjhoncv.github.io/dashboard/) 

## Features

- Version de Express JS 4
- Version de Typescript 4
- Version de MySQL 5.6

### Step 1
Up MySQL
```sh
make db
```

### Step 2
Up Express JS - API
```sh
make start
```

### Extras
Down Express JS - API
```sh
make stop
```

Dependencies NPM
```sh
make install
```
